# 🎬 Task Forge · 功能演示 GIF 录制完整指南

> **工具推荐**：[ScreenToGif](https://www.screentogif.com/)（免费，精确帧控制）或 [LICEcap](https://www.cockos.com/licecap/)  
> **分辨率**：录制区域建议 **1280×800**，输出 GIF 缩至 **640px** 宽，控制文件体积  
> **帧率**：12–15 fps 足够流畅，降低体积  
> **时长**：每段 GIF 建议 **10–25 秒**，突出核心操作，不要空白等待

---

## 📋 录制前准备

```bash
# 1. 清空演示库，重新植入干净演示数据
cd Task-Forge
python -c "
import os, shutil
db_path = 'data/task_forge.db'
bak_path = 'data/task_forge.db.bak'
if os.path.exists(db_path): os.remove(db_path)
if os.path.exists(bak_path): os.remove(bak_path)
print('已清空数据库，下次启动将自动植入演示数据')
"

# 2. 启动应用（自动植入演示数据）
python src/main.py
```

- 窗口最大化，建议 **1400×900** 以上分辨率录制  
- 关闭其他占用 CPU 的程序，避免录制卡顿  
- 鼠标速度调慢，操作要清晰可辨
